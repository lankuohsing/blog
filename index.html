Posted on 2018-02-01

Introduction to RNN

In a feedforward neural network, we assume that all inputs (and outputs) are independent of each other. Therefore, feedforward neural networks can’t remember the recent past information of training, and become power less dealing with sequence models. Recurrent neural network(RNN) was proposed to address this issue (Jordan et al. 1986, Elman et al. 1990). RNNs make use of sequential information by maintaining in their hidden units a ‘state vector’ that implicitly contains information of all the past elements of the sequence. The input of an RNN model is an sequence. The model processes one element in the input sequence at one time step. After computation of one time step, the newly updated state vector is passed down to the next time step to facilitate the computation of thee next element. Here is what a typical RNN model looks like:

A recurrent neural network with one hidden unit(left) and its unfolding version in time involved in its forward computation(right). The state $s_t$ in hidden unit at time $t$ get inputs from input $x_t$ at time $t$ and previous state $s_{t-1}$ at time $t-1$. In this way, an RNN model can map an input sequence $\left\{ x_t \right\}$ into an output sequence $\left\{ o_t \right\}$, with each $o_t$ depending on all the previous $x_{t’}$(for $t’\leq t$)

However, theoretical and empirical evidence has shown that it’s difficult for RNNs to learn long-term dependencies (Bengio et al, 1994), because the gradients will vanish or explode when trained with gradient-based methods. To resolve this problem, a much more complicated network was proposed, named Long-Short Term Memory (LSTM ) network.

LSTM

All recurrent neural networks have the form of a chain of repeating modules of neural network. In standard RNNs, this repeating module will have a very simple structure, such as a single tanh layer.

LSTMs also have this chain-like structure, but the repeating module is different from that of standard RNNs, and much more complicated. Instead of having a single neural network layer, there are four, interacting in a very special way.

LSTM is able to learn for how long it should memorize the old information, when to forget, when to make use of the new data, and how to combine the old memory with new input, carefully regulated by structures called gates.

Detailed introduction to the LSTM diagram above is shown in Understanding LSTM Networks.

LSTM Variants

A variation is to use coupled forget and input gates. Gated Recurrent Unit(GRU) is an instance of this. It combines the forget and input gates into a single “update gate.” It also merges the cell state and hidden state.
